программа курса

Модуль 1
(Введение)

Познакомимся с первыми тегами HTML, их классификацией. Изучим способы подключения стилей CSS к веб-странице, первые стилевые свойства.

Научимся вставлять изображения на веб-страницы и работать с гиперссылками.

Узнаем как работать с цветом в CSS. Разберемся с единицами измерения в CSS.

Изучим основные теги и стили для работы с текстом. Научимся создавать списки HTML, подключать и стилизовать шрифты.

Модуль 2
(Верстка)

Научимся верстать таблицы.

Узнаем, как браузер рассчитывает размеры каждого элемента (блока) на веб-странице и определяет их местоположение. Узнаем, как блоки сайта влияют на положение и размер друг друга. Изучим строение блоков. Научимся создавать макеты сайтов, задавая характеристики блоков.

Научимся задавать позицию элементов на веб-странице с помощью свойств position и z-index.

Научимся использовать иерархические селекторы CSS и создавать сайты, которые корректно отображаются в браузерах любых устройств (десктоп, планшет, смартфон).

Научимся создавать адаптивные макеты веб-страниц с помощью современных технологий Flex и Grid. Узнаем как делать сайты, корректно отображающиеся в различных реализациях и поколениях интернет-браузеров.

Модуль 3
(Анимация и взаимодействие)

Научимся анимировать элементы веб-страницы с помощью CSS. Узнаем, как подлючать медиафайлы к веб-страницам.

Познакомимся с языком программирования Java Script (JS) и объектной моделью веб-документа (DOM). Научимся поключать JS к веб-страницам и использовать его возможности для создания мобильного меню и слайдеров.

Применяем знания на практике комплексно. Создаем веб-сайты самостоятельно!
Ученики, оформившие подписку на наш курс в Boosty уровня "Квантовый компьютер", получают консультационную поддержку в закрытом чате.
